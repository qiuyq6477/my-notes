-   ﻿﻿Complicated Tool GUI
	-   Immediate Mode
		- 直接调用绘制控件的API
		- 直接简单，可以快速迭代
		- 控件本身不包括任何数据和状态
		- 性能不好
		- 不好扩展
		- ![[Pasted image 20230120182120.png]]
	-   Retained Mode
		- 调用graphics library的API，让这个library去绘制，library内部可以做优化，可以统一处理。如果ui没变化，可以不用去绘制，效率更高。
		- 控件保存有自己的状态和数据
		- 更好的可扩展性，更好的性能
		- 但是对于开发者，编码会比较复杂，需要在gui和logic之间做同步
		- ![[Pasted image 20230120182028.png]]
		- 可以应用设计模式
